# Dallas-DS1284-to-DS1286-Adapter
This is a small PCB that can be glued to the back of a Dallas DS1284 to turn it into a DS1286, providing a user replaceable backup battery.

You will need to bend up pins 2, 3, 21, and 25 to solder them to the adapter PCB, as per the original Dallas design.
Be sure to carefully loop the pins 2 and 3 over the guard trace so they don't short out!

When you get PCB's made, there are two castellated edges, so be sure to apply that option when placing your order.

There are only two parts on this PCB:

X1 - FKFSREIBM0.032768-T3 or equivalent (any decent 32.768kHz crystal can be used)
BT1 - BU1632SM-JJ-GTR or equivalent SMD CR1632 battery holder

This uses a standard CR1632 coin cell. To retain memory, only replace the cell while power is applied to the chip.
